Getting Started with Phastats

Learn how to install and use Phastats for comprehensive FASTQ quality analysis.

Installation

Option 1: Install from PyPI (Recommended)

pip install phastats

Option 2: Install from Source

git clone https://github.com/your-username/phastats.git
cd phastats
pip install -e .

Dependencies

Phastats requires the following Python packages:

  • matplotlib >= 3.3.0
  • numpy >= 1.19.0
  • pandas >= 1.1.0
  • scipy >= 1.5.0

Quick Start

Command Line Usage

Basic usage with default settings:

phastats input.fastq

Specify output directory:

phastats input.fastq -o output_directory

Generate specific plots only:

phastats input.fastq --plots gc length quality

Python API Usage

from phastats.core import analyze_fastq

# Basic analysis
results = analyze_fastq("input.fastq")

# Advanced analysis with custom parameters
results = analyze_fastq(
    "input.fastq",
    output_dir="my_analysis",
    quality_threshold=25,
    enabled_plots={'gc', 'length', 'quality'}
)

Web Interface

Phastats includes a web interface for easy file upload and analysis.

Running the Web Interface

cd phastats-web
pip install -r requirements.txt
python app.py

Then visit http://localhost:5000 in your browser.

Output Files

Phastats generates several output files:

  • phastats_report.html - Interactive HTML report
  • phastats_statistics.json - JSON format statistics
  • plots/ - Directory containing all generated plots

Analysis Features

Quality Score Analysis

  • Per-base quality score distributions
  • Overall quality statistics
  • Low-quality base identification

Sequence Length Analysis

  • Length distribution histograms
  • N50 calculation
  • Mean and median lengths

GC Content Analysis

  • Per-sequence GC content
  • GC distribution plots
  • GC bias detection

Per-Base Content

  • Base composition across read positions
  • Sequence complexity analysis
  • Adapter contamination detection

Configuration Options

Phastats supports various configuration options:

Quality Thresholds

  • --quality-threshold - Minimum quality score (default: 20)
  • --high-quality-threshold - High quality threshold (default: 30)

Performance Options

  • --chunk-size - Processing chunk size (default: 10000)
  • --skip-perbase - Skip per-base analysis for faster processing
  • --skip-n50 - Skip N50 calculation

Troubleshooting

Common Issues

Permission denied: Use chmod +x phastats on Unix systems
Missing dependencies: Install with pip install -r requirements.txt
Large files: Use --chunk-size to adjust memory usage
