Advanced FASTQ Quality Analysis

Phastats is a professional bioinformatics tool that provides comprehensive quality control analysis for FASTQ sequencing data with modern visualizations and detailed reporting capabilities.

Comprehensive Analysis Features

Phastats provides all the essential quality control metrics you need for FASTQ data analysis, with modern visualizations and detailed reporting.

Quality Distribution

Comprehensive quality score analysis with per-base quality metrics and distribution plots.

Length Analysis

Sequence length distribution analysis with N50 calculations and statistical summaries.

GC Content

GC content analysis with per-sequence distribution and bias detection.

Quick Installation

Command Line Tool

# Install from PyPI
pip install phastats

# Or install from source
git clone https://github.com/somtoik/phastats.git
cd phastats
pip install -e .

Web Interface

# Clone repository
git clone https://github.com/somtoik/phastats.git
cd phastats/phastats-web

# Install dependencies
pip install -r requirements.txt

# Run the web app
python app.py
